# ---PROJECT - Library Management System ---

# DESCRIPTION
The Library Management System is a software project designed to manage and organize library activities efficiently. It helps librarians and users to maintain records of books, issue/return transactions, user details, and availability status.
This project reduces manual effort, avoids errors, and makes the library workflow faster and more accurate.

# ALGORITHM
1. Start
2. Add Book
      -Input book title
      -Create a record {title, borrowed=False}
      -Add it to the library list
3. Display Books
      -For each book in the list:
      -Show title and status (Available / Borrowed)
4. Borrow Book
      -Check if selected book is not borrowed
      -If available → mark as borrowed
      -Else → show "Already borrowed
5. Return Book
      -Check if selected book is borrowed
      -If yes → mark as returned
      -Else → show "Not borrowed"
6. End

# FEATURES 
The system allows users to:
-Add, update, delete, and search books
-Maintain student/user records
-Issue and return books with proper validation
-Track due dates and available copies
-Store data securely and systematically
